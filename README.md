# FAHM Mammogram Classifier

FAHM is an AI-assisted mammogram screening system developed using EfficientNet-B0 and GradCAM explainability.

## Features

* Mammogram image classification
* Malignant vs Benign/Normal prediction
* Confidence score reporting
* GradCAM visualisation
* Privacy-focused architecture
* Streamlit web interface

## Technology Stack

* Python
* PyTorch
* EfficientNet-B0
* Streamlit
* GradCAM

## Clinical Disclaimer

This application is intended for research and educational purposes only.

It is not a medical device and must not be used as a substitute for professional clinical judgement. All results should be reviewed by a qualified radiologist.

## Privacy

* Images processed in memory only
* No patient data stored
* No image retention after inference

## Deployment

Hosted publicly using Hugging Face Spaces.
